projects

The Volcano List Editor is a JavaFX application that allows a volcanologist to keep track of volcanoes and any associated eruptions. The user can add/delete volcanoes to/from a list, as well as add/delete eruptions to an individual volcano. Data is stored in two seperate CSV files, one for the volcanoes and one for the eruptions, in order to save data between instances.

The user can view a chart of a particular volcano's eruptions using the View Line Charts tab. This displays all of the eruptions for a chosen volcano on a line graph, with magnitude on the y-axis and year of eruption on the x-axis.

This project was completed in CMPT 305 (Object Oriented Programming) at MacEwan University in Edmonton, Alberta.

The Edmonton Property Assessment Application is a JavaFX project that allows the user to look information about Edmonton properties. Property data is loaded into the program from a CSV file. The user can then search through the data via address, neighbourhood, assessment class, or account number.

The statistics of the returned properties are displayed as long as there are more than one returned items.

This project was completed in CMPT 305 (Object Oriented Programming) at MacEwan University in Edmonton, Alberta.

MR RoBug was made in 48 hours as part of the Game Maker's Toolkit Game Jam in June of 2021. This was my first game jam project.

The purpose of the game is to get both your characters through the locked door. The robot bug (RoBug) follows the alien's movement but cannot jump on it's own.

The project was made using the Godot game engine. Art and sound were obtained from Kenney.nl, soundimage.org, and freesound.org.
